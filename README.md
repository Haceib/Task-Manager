# Interactive To-Do / Task Manager

## Project Info
A responsive Task Manager web app built with **React** and **TailwindCSS**. Manage daily tasks with add, edit, delete, and complete features, including persistent storage using localStorage.

---

## Features

- Add, edit, and delete tasks  
- Mark tasks as complete/incomplete  
- Filter tasks by status (All, Completed, Pending)  
- Persist tasks in localStorage  
- Optional enhancements:
  - Drag-and-drop task reordering  
  - Task categories and priority levels  
  - Dark/Light mode toggle  

---

## UI / Design

- Clean, modern, fully responsive design using TailwindCSS  
- Smooth hover effects and transitions  
- Mobile-friendly layout  

---

## Tech Stack

- React.js  
- JavaScript / TypeScript  
- TailwindCSS  
- LocalStorage API
